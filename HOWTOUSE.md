# Como Usar o Loot Logger

Guia rapido para usuarios do Loot Logger.

## Primeiro Uso

### 1. Instalacao do Npcap

Antes de usar o Loot Logger, voce precisa instalar o Npcap:

1. Acesse: https://npcap.com/#download
2. Baixe o instalador
3. Execute como administrador
4. Marque a opcao "Install Npcap in WinPcap API-compatible Mode"
5. Conclua a instalacao

### 2. Ativacao da Licenca

1. Execute o Loot Logger
2. Na tela de ativacao, digite sua chave de licenca
3. Clique em "Ativar Licenca"
4. Aguarde a validacao

Sua licenca sera salva automaticamente para os proximos usos.

## Usando o Loot Logger

### Iniciando a Captura

1. Abra o Albion Online
2. No Loot Logger, clique no botao "Iniciar" (verde)
3. O status mudara para "Albion Detectado" quando o jogo for encontrado
4. Todos os loots serao exibidos automaticamente

### Parando a Captura

1. Clique no botao "Parar" (vermelho)
2. A captura sera interrompida

## Filtros

### Filtro por Tier

No painel esquerdo, voce pode marcar/desmarcar os tiers que deseja ver:
- T4, T5, T6, T7, T8
- Opcao "Raros" para mostrar apenas itens raros

### Filtro por Jogador

1. Digite o nome do jogador no campo de busca
2. Ou clique no nome do jogador na lista
3. Use "Todos" para mostrar todos os jogadores
4. Use "Nenhum" para limpar a selecao

## Configuracoes

Clique no icone de engrenagem para acessar as configuracoes.

### Idioma

Selecione o idioma dos nomes dos itens:
- PT-BR (Portugues)
- EN-US (Ingles)

### Discord Webhook

Para receber notificacoes no Discord:

1. No Discord, va em Configuracoes do Servidor > Integracoes > Webhooks
2. Crie um novo webhook
3. Copie a URL do webhook
4. Cole no campo "Discord Webhook" do Loot Logger
5. Ative a opcao "Ativado"
6. Clique em "Testar" para verificar
7. Clique em "Salvar"

## Exportacao

### Exportar Loots

1. Clique em "Exportar"
2. Escolha o formato (JSON ou CSV)
3. Selecione o local para salvar
4. Clique em "Salvar"

## Solucao de Problemas

### "Albion Nao Detectado"

- Verifique se o Albion Online esta aberto
- Execute o Loot Logger como administrador
- Verifique se o Npcap esta instalado corretamente

### Licenca Invalida

- Verifique se digitou a chave corretamente
- Verifique sua conexao com a internet
- Entre em contato com o suporte

### Loots Nao Aparecem

- Verifique se clicou em "Iniciar"
- Verifique os filtros de tier
- Certifique-se de que o Albion foi detectado

## Suporte

Para duvidas ou problemas, entre em contato com o suporte.
